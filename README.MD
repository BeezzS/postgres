Este repositório foi criado com o objetivo de armazenar **exercícios e práticas de estudo em PostgreSQL**, focando no aprendizado de consultas SQL e conceitos básicos de bancos de dados relacionais.

- **PostgreSQL** – Banco de dados relacional utilizado nos estudos  
- **DBeaver** – Ferramenta utilizada para gerenciar o banco de dados e executar as consultas SQL
