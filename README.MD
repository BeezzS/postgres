Este repositório reúne exercícios e práticas realizadas durante um treinamento de **PostgreSQL**, com foco no aprendizado de conceitos fundamentais de bancos de dados relacionais e na construção de consultas SQL.

Ao longo do curso, foram realizadas **58 atividades práticas**, explorando desde os primeiros passos até consultas mais avançadas.

## 🧠 Conteúdos abordados

- Primeiros passos com PostgreSQL
- Operações **CRUD**
  - Create
  - Read
  - Update
  - Delete
- Consultas com **filtros**
- **Relacionamentos** entre tabelas
- Uso de **CASCADE** em chaves estrangeiras
- Construção e execução de **consultas SQL**
- Consultas mais avançadas com organização e manipulação de dados

## 🛠 Ferramentas utilizadas

- **PostgreSQL** – Sistema de gerenciamento de banco de dados relacional
- **DBeaver** – Utilizado como SGBD para executar consultas, visualizar tabelas e gerenciar o banco de dados

## 📖 Documentação

A documentação oficial do PostgreSQL foi utilizada como referência durante os estudos:

- https://www.postgresql.org/docs/

## 🎯 Objetivo

O objetivo deste repositório é consolidar conhecimentos em **SQL e PostgreSQL**, praticando conceitos fundamentais utilizados no desenvolvimento de sistemas e na manipulação de dados.
